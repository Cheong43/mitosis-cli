# 台北美食分類指南

## 一、夜市經典美食

| 美食名稱 | 簡介 | 推薦吃法 |
|---------|------|---------|
| 蚵仔煎 | 將鮮蚵搭配太白粉漿煎製，外皮酥脆內餡軟嫩 | 淋上特製醬汁，配上豆芽菜 |
| 臭豆腐 | 發酵黃豆製成，油炸後外酥內軟 | 搭配泡菜與蒜泥醬汁 |
| 大腸蚵仔麵線 | 傳統台式麵線，濃稠湯底配上大腸與鮮蚵 | 加醋與辣椒調味 |
| 雞排 | 厚實雞肉裹粉油炸，撒上胡椒鹽 | 切塊方便食用，外酥內多汁 |
| 珍珠奶茶 | 手搖茶飲標誌性飲品，Q彈粉圓搭配香醇奶茶 | 大杯去冰微糖最佳 |
| 藥燉排骨 | 中藥材熬煮的排骨湯，溫補養生 | 冬季限定的暖心選擇 |
| 豆花 | 軟嫩豆花配上糖水與配料 | 冬天熱食，夏天冰食皆宜 |
| 剉冰 | 細緻冰品基底搭配多種配料 | 芒果、芋頭、花生任選 |
| 花生冰淇淋 | 傳統碳烤花生磨成醬，配上香草冰淇淋 | 潤餅皮包裹，獨特口感 |
| 胡椒餅 | 碳火烤制的香酥餅，內餡為絞肉與蔥 | 現烤出爐最香 |
