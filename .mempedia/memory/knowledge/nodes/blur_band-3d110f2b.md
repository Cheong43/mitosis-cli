# 模糊乐队（Blur）

## Facts
- 英国摇滚乐队，1989年成立于伦敦科尔切斯特
- 成员包括主唱代蒙·阿尔本（Damon Albarn）、贝斯手亚历克斯·詹姆斯（Alex James）、吉他手格雷厄穆·考可森（Graham Coxon）、鼓手戴夫·朗特别里（Dave Rowntree）
- 开创Brit-pop风格，打破90年代初期舞曲对乐坛的垄断
- 通过1994年专辑《Parklife》奠定英国流行文化主导地位

## Data
- 处女作：1990年单曲《She's So High》
- 代表专辑：《Leisure》(1991),《Modern Life Is Rubbish》(1993),《Parklife》(1994) 白金销量
- 关键转折：1997年专辑《Blur》转向美国独立摇滚风格
- 中国百度百科条目更新时间：2026年隐私政策标注

## History
- 1989年组建时名为Seymour
- 1990年签约EMI子公司Food唱片
- 1995年获英国BPI四项大奖
- 1996年面临解散危机后转向美国市场
- 1997年《Song 2》单曲在美走红后反哺英国市场

## Relations
- 与绿洲（Oasis）乐队存在竞争关系
- 被达米安·赫斯特（Damien Hirst）称为披头士以来英国最棒乐队
