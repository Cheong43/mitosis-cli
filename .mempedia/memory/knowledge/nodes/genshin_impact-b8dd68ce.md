# 原神 (Genshin Impact)

## Facts
- 3D角色扮演开放世界冒险游戏
- 提瓦特大陆包含七种元素（风/岩/雷/草/水/火/冰）
- 每个元素对应一个执政官（神明）和国家
- 支持PS4/PS5/PC/移动端等多平台
- 提供「神之眼」作为元素操控核心道具
- 与崩坏系列存在世界观联动（如崩坏3的奥托提及坎瑞亚）

## Data
- 开发商：米哈游（2011年成立）
- 全球同步发行时间：2020年9月28日（PS4）
- 中国公测时间：2020年10月21日
- 主角设定：名为「旅行者」的漂泊者
- 七神对应国家：蒙德/璃月/稻妻/须弥/枫丹/纳塔/至冬

## History
- 2017年1月公布首个PV「VISION」
- 2020年6月开启预下载
- 2021年6月与崩坏3首次联动
- 2024年与崩坏：星穹铁道联动提瓦特羽翼

## Relations
- 开发商关联：崩坏系列、未定事件簿
- 世界观关联：坎瑞亚（至冬国地下古国）
- 同类型游戏：塞尔达传说、对马岛之魂
